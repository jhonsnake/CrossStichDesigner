# Prompt para Generador de Patrones de Punto de Cruz

## Descripción del Proyecto

Crea una aplicación web completa para generar patrones de punto de cruz a partir de imágenes subidas por el usuario. La aplicación debe ser intuitiva, visualmente atractiva y especialmente diseñada para ser fácil de usar por personas mayores.

## Características Principales

1. **Subida de imágenes**: Permitir a los usuarios subir imágenes en formato JPG o PNG.
2. **Configuración personalizable**:
   - Dimensiones del patrón (ancho y alto en puntos)
   - Tamaño de la tela (ancho y alto en cm)
   - Tipo de tela (Aida 14, Aida 16, etc.)
   - Tipo de hilo (DMC, Anchor, J&P Coats)
   - Control del número de colores (automático o específico)
3. **Visualización del patrón**:
   - Vista del patrón con colores y símbolos
   - Controles de zoom
   - Opción para mostrar/ocultar colores y símbolos
4. **Cálculo de materiales**:
   - Lista de hilos necesarios con códigos de color
   - Cantidad de puntos por color
   - Estimación de longitud de hilo y número de madejas
5. **Exportación**:
   - Exportar el patrón a PDF con instrucciones detalladas
   - Incluir guías de corte para patrones grandes

## Diseño y Estilo

- Utiliza una paleta de colores pastel suave y agradable
- Interfaz limpia y espaciosa con texto grande y legible
- Botones grandes con iconos descriptivos
- Diseño responsive que funcione bien en tablets y computadoras

## Tecnologías a Utilizar

- Next.js con App Router
- React para la interfaz de usuario
- Tailwind CSS para estilos
- shadcn/ui para componentes de interfaz
- jsPDF para la generación de PDFs

## Estructura de Archivos

1. **Componentes principales**:
   - `ImageUploader.tsx`: Componente para subir imágenes
   - `PatternGenerator.tsx`: Visualización del patrón generado
   - `ThreadCalculator.tsx`: Cálculo de hilos y materiales

2. **Utilidades**:
   - `image-processor.ts`: Procesamiento de imágenes y generación de patrones
   - `thread-colors.ts`: Base de datos de colores de hilos

3. **Página principal**:
   - `app/page.tsx`: Página principal que integra todos los componentes

## Detalles de Implementación

### Base de Datos de Colores

Implementa una base de datos completa de colores de hilos para:
- DMC (amplia gama de colores)
- Anchor
- J&P Coats

Cada color debe incluir:
- ID único
- Código del fabricante
- Nombre del color
- Valor hexadecimal
- Valores RGB

### Procesamiento de Imágenes

El algoritmo de procesamiento debe:
1. Redimensionar la imagen al tamaño de cuadrícula deseado
2. Analizar cada píxel y asignar el color de hilo más cercano
3. Permitir limitar el número de colores utilizados
4. Generar una matriz de colores para el patrón

### Exportación a PDF

El PDF generado debe incluir:
1. Información de configuración utilizada
2. Lista completa de colores con muestras visuales
3. Leyenda de símbolos
4. Instrucciones detalladas para el bordado
5. El patrón dividido en secciones si es grande, con guías de corte
6. Información sobre cómo unir las secciones

## Características Adicionales

- Implementa un sistema para limitar el número de colores, permitiendo al usuario elegir entre modo automático o un número específico de colores
- Agrega información sobre la complejidad del patrón basada en el número de colores
- Incluye consejos útiles para principiantes en las instrucciones

## Consideraciones de Accesibilidad

- Asegúrate de que todos los elementos tengan suficiente contraste
- Incluye textos alternativos para imágenes
- Diseña pensando en usuarios con poca experiencia tecnológica
- Utiliza fuentes legibles y de buen tamaño

## Ejemplo de Flujo de Usuario

1. El usuario sube una imagen
2. Configura las dimensiones del patrón, tipo de tela e hilo
3. Selecciona si quiere usar todos los colores disponibles o limitar a un número específico
4. Genera el patrón
5. Explora el patrón usando los controles de zoom y visualización
6. Revisa la lista de materiales necesarios
7. Descarga el PDF con el patrón completo e instrucciones

Este proyecto debe ser una herramienta completa y fácil de usar para cualquier persona interesada en convertir sus imágenes favoritas en patrones de punto de cruz, con especial atención a la usabilidad para personas mayores.
